Defi CSS 06 - Les Media Queries

Ecrivez le code CSS nécessaire pour que le carré présent sur la page soit :

  • Rouge : Quand la résolution est inférieure à 481px
  • Vert : Quand la résolution est inférieure à 769px
  • Bleu : Quand la résolution est inférieure à 1025px
  • Orange : Quand la résolution est inférieure à 1201px
  • Violet : Quand la résolution est égale à 1201px ou plus
