Defi JS - 12

Objectif : Lorsque l'utilisateur scroll sur la page web, une navbar qui est cachée au chargement de la page apparaît.

Consigne : Dans le CSS, vous metterez une position Top à votre navbar de -56px, l'idée de ce défi, c'est d'obtenir le nombre de pixels dont le contenu de l'élément html a défilé vers le haut, et de mettre une condition qui passe la position Top de la navbar à 0 quand le défilement atteint une certaine hauteur.
